Qt SDK Repo Chooser

If you are using the online Qt 5 SDK but downloads are slow and/or you frequently get disconnected, this tool might help. You might have been assigned a mirror that is performing poorly. The Qt SDK Repo Chooser helps you select a non-default mirror, which might give you a better download experience.

Ideally, the causes of this issue should be solved upstream. Please add your thoughts, ideas, and votes to the reports at QTIFW-5 and QTIFW-441. In the meantime though, I hope this tool is useful for you to get started with Qt.

Usage Instructions

  1. Go to http://www.qt.io/download-open-source/ and click "View All Downloads".

  2. Choose a random file and click "(Info)". At this stage, it doesn't matter which file you choose; we simply want to check the servers' download speeds.

  3. Choose a mirror and download the file. Keep an eye on your download speed for about 15 seconds. If you are satisfied with the download speed, cancel the download and go to step #4. Otherwise, try another mirror.

  4. Download and extract the Qt SDK Repo Chooser. Launch the QtSdkRepoChooser executable.

  5. Select the good mirror that you found in step #3, and click "Use Mirror".

  6. When the Local Repo URL appears, copy it (Ctrl+C on Windows/Linux, Cmd+C on macOS).

  7. Launch your online Qt installer. After you log into your Qt Account (or skip the login), click "Settings".

  8. Under the "Repositories" tab, deselect "Qt online root repository". (Note: If you already have an existing installation, there will be other repositories listed. Don't deselect these)

  9. Add a User Defined Repository. Paste the URL from step #6.

  10. Download Qt. Happy coding!

IMPORTANT!: When a new version of Qt is released, you will need to re-run steps #5 and #6 in the Qt SDK Repo Chooser to update your local repo with details of the new packages.

Alternative Instructions

If you can't use this tool for whatever reason, see README.md in the project root folder for an alternative way to choose your repo.
